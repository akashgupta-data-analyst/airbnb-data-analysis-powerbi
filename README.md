# Airbnb Data Analysis (Power BI Project)

---

## Project Overview

This project focuses on analyzing Airbnb listing data to uncover insights related to pricing, customer preferences, and host performance.

An interactive Power BI dashboard was created to help stakeholders understand trends and make data-driven decisions.

---

## Problem Statement

Airbnb hosts and platforms often struggle to:

* Identify optimal pricing strategies
* Understand customer booking patterns
* Track performance across different locations

This project solves these problems by transforming raw data into meaningful visual insights.

---

## Key KPIs

* Total Listings
* Average Price
* Total Reviews
* Reviews per Month
* Active Hosts

---

## Key Insights

* Manhattan and Brooklyn dominate in total bookings and listings
* Certain neighborhoods have significantly higher pricing trends
* Peak reviews observed during mid-year indicating seasonal demand
* Entire home/apartment listings generate higher revenue compared to shared rooms

---

## Dashboard Features

* KPI cards for quick overview
* Neighborhood-wise booking analysis
* Monthly trend analysis
* Price comparison by room type
* Top hosts by reviews
